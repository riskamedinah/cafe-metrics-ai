**Cafe Metrics** adalah platform dashboard analitik modern yang dirancang untuk mengelola data bisnis, metrik penjualan, dan manajemen produk. Project ini di-slicing secara **pixel-perfect** dari Figma dengan fokus pada akurasi layout, tipografi yang tajam, dan struktur kode yang bersih (*clean code*).
---
## 🛠️ Tech Stack & Style Guide
Aplikasi ini dibangun menggunakan teknologi *front-end* modern dengan konfigurasi berikut:
*   **Framework:** React (Vite)
*   **Styling:** Tailwind CSS v4 (Menggunakan fitur @import "tailwindcss"; dan CSS Variables modern)
*   **Typography:** Geist Variable Font (@fontsource-variable/geist) untuk tampilan teks yang bersih dan profesional.
*   **Icons:** Lucide React (lucide-react)
*   **Charts:** Recharts (Untuk visualisasi data grafik yang interaktif)
### 🎨 Custom Design Tokens (CSS Variables)
Sistem pewarnaan dikendalikan secara terpusat melalui variabel :root berikut untuk menjaga konsistensi dengan *style guide* Figma:
*   **Primary (Brand):** #3B5BDB (Hover: #3451C7)
*   **Neutral Grayscale:** Menggunakan skala warna khusus dari var(--color-neutral-50) (#F4F5F7) hingga var(--color-neutral-900) (#373742).
